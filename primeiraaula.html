Adobe DreamWeaver

O que é e o que fazemos com o software DreamWeaver?

O Dreamweaver é um programa de edição de paginas da internet, onde se pode trabalhar com imagens, textos, e muitos outros elementos para a Web. O programa permite a criação e edição de paginas na internet, sendo em alguns aspectos, muito parecido com programas de criação de layouts. Com este programa não é necessário criar um códigos de programação, acelerando assim o tempo de produção de sites.

Com o programa Dreamweaver podemos desenvolver e editar páginas na Web, importar ativos de bibliotecas da Creative Cloud e do Adobe Stock, editar propriedades de texto e imagem, verificar sintaxe em tempo real, gerar dicas de código, publicar sites em servidores externos, realizar pequenas alterações rapidamente e vê-las refletidas imediatamente no servidor, e muito mais. O Dreamweaver pode ser trabalhado em diversas linguagens de programação, dentre elas estão HTML, CSS, JavaScript, PHP, Ação Script, Páginas do Servidor Ativo (ASP), C#, XHTML, XML, Java, etc.

Como trabalhamos ou fazemos?

Ele permite criar sites escrevendo os comandos ou até construir o esqueleto de um site rapidamente e sem a necessidade de escrever uma única linha de código.

Explique o HTML

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento. O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

O que são as TAGs, descreva exemplos?

As tags do HTML são elementos que indicam ao navegador como estruturar uma página web. São como etiquetas que marcam o início e o fim de conteúdos, como parágrafos, imagens, links e títulos. Alguns exemplos são: p = Cria um parágrafo, h1 = Define o título maior, h6 = Define o título menor, input = Cria um campo de entrada interativo, body = Onde fica o conteúdo do texto, imagem e vídeos.

Explique o CSS?

A linguagem Cascading Style Sheet, chamado de CSS, é usada para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página. Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos. Também pode criar tabelas, usar variações de layouts, ajustar imagens para suas respectivas telas e assim por diante.

Explique UTF-8 e para que usamos

O UTF-8 é uma codificação de caracteres que serve para converter caracteres em uma sequência de bits, de modo que o computador possa entendê-los, usado também para definir o site em português Brasil. É o padrão de codificação de caracteres mais usado na internet. Ele pode ser utilizado em páginas da web, plataformas baseadas em *nix, linguagens de programação como Python e JavaScript, sistemas Linux, formatação de textos, QR-Code, etc.
